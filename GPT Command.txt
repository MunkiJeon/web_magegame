간단한 웹 게임을 만들려고 해,  뼈대 코드 만드는 거 도와줘
이 게임은 Player의 캐릭터가 출발 지점에서 골인 지점까지 제한 시간 안에 탈출 지점을 찾아 탈출하는 게임이야,
기본적으로 필요한 기능은
- 게임 시작, 다시 시작, 제한 시간
- 재한 시간 안에 골인 지점 도착 시 개임 중지
- 재한 시간 초과 시 개임 중지 
- 매번 개임 시작 혹은 다시 시작 버튼 누르면 랜덤 한 위치에 캐릭터와 골인 지점 배치
- 골인 지점 도착 시 도착까지 걸린 시간이 표시되어 있는 "다시 시작"과 "끝내기" 버튼이 있는 팝업
- 방향키 혹은 wasd로 캐릭터 이동
